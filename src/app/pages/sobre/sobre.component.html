Ideia do site

Esse site foi construído com o objetivo de trazer alguns dados sobre os mais diversos filmes existentes no mundo. Dados como:

  • Avaliação;
  • Banner do filme;
  • Data de lançamento;
  • Quantidade de avaliações;
  • Título;
  • Um breve resumo sobre o filme.

Origem dos dados

Essas informações vêm através da The Movie Database API (TMDB API). Esta API fornece todos os dados sobre os filmes, fornece também a lista de gêneros e a lista de opções de idiomas. A TMDB API forneces mais uma imensa quantidade de dados, mas decidi usar somente esses citados para construção desse site.

Como usar o site

O site conta com duas páginas:

  • Home: página que contêm a melhor parte do site, é nela que você irá interagir com a API. Para usar é muito simples, basta seguir o seguinte passo a passo:

    1. Selecione o gênero de filme que deseja buscar;
    2. Selecione o idioma em que deseja que os dados dos filmes sejam apresentados;
    3. Selecione a ordem na qual deseja que os filmes sejam apresentados;
    4. Clique no ícone da lupa e veja a magia acontecer :)

    Após os filmes serem apresentado é possível clicar no filme que deseja para obter mais informações sobre ele. Por fim, essa pagina possui paginação, ou seja, é possível visualizar todos os filmes do gêneros escolhido fazendo a navegação disponibilizada antes e depois dos cards dos filmes.

  • Sobre: página que você se encontra lendo no momento. Ela apresenta informações sobre o site, origem dos dados e um pequeno tutorial de como usar o site.
